# Requisitos Funcionais

- O sistema deve utilizar MySQL.
- O sistema deve possuir interface intuitiva e de fácil utilização.
- O sistema deve ser desenvolvido na linguagem Java.
- O sistema deve funcionar 24 horas por dia.
- O sistema deve permitir acesso simultâneo de vários usuários.
- O sistema deve seguir arquitetura REST.
- O sistema deve utilizar o framework Spring Boot.
- O sistema deve garantir a persistência dos dados.
- O sistema deve validar os dados recebidos nas requisições.
- O sistema deve possuir boa performance e tempo de resposta adequado.
